Block Elements:

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.It always takes up the full width available stretches out to the left and right as far as it can

Inline Elements:

An inline element does not start on a new line.An inline element only takes up as much width as necessary. Have you seen this amazing video on YouTube?


Inline elements are used within block-level elements to style or format specific parts of the content.They don’t start on a new line and only take up as much width as necessary for example include span,em etc..

Elemnts types:

STRONG: The tag is used to define text with strong importancethe content inside is typically displayed in bold.

QUOTES: The Q tag defines a short quotes. Browsers normally insert quotation marks around the quotation.

ABBREVIATIONS: The tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
